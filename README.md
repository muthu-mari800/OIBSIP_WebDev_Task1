# OIBSIP_WebDevelopment_Task1 – ExpressVPN Landing Page

## Project Title
ExpressVPN Responsive Landing Page (UI Clone)

## Project Description
This project is a responsive landing page inspired by the official ExpressVPN website.  
It is created using **HTML5 and inline, internal CSS** to demonstrate front-end web development and UI design skills.  
The page highlights VPN features, global server coverage, security benefits, and call-to-action sections.



## Objective
- To design a professional VPN landing page
- To practice HTML structure and CSS positioning
- To create a visually appealing and informative webpage
- To understand layout design using absolute and relative positioning



## Technologies Used
- HTML5  
- CSS3 (Inline ,Internal CSS)  
- Responsive design using viewport units (vw, vh)



## Project Structure
OIBSIP_WebDevelopment_Task1/
│
├── index.html

## Features
- Full-screen hero section with background image
- Navigation header with language option
- Call-to-action buttons
- Informative sections about VPN security and coverage
- Grid-based feature highlights
- Global server statistics section
- Clean and professional UI design



## Steps Performed
1. Designed the main layout using HTML div structure
2. Applied inline CSS for styling and positioning
3. Used background and illustrative images to enhance UI
4. Implemented responsive sizing using viewport units
5. Organized content into multiple sections for better readability
